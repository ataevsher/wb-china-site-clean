# Project overview

This project is a B2B service website focused on China sourcing, logistics, and marketplace support for Russia, including Wildberries and Ozon.

## Main goal
Create a high-conversion, premium-looking website that builds trust and generates inbound leads from businesses interested in:
- sourcing products from China
- logistics and delivery to Russia
- support with marketplaces such as Wildberries and Ozon
- import/export consulting
- warehouse and fulfillment solutions

## Current stage
The current website is primarily a landing page. It should be improved first as a strong, polished conversion-oriented landing page. Later it may evolve into a multi-page website.

## Languages
The website should support:
- Russian (primary)
- English
- Chinese

All visible text in all language versions must be fully translated consistently. No mixed-language leftovers are allowed.

## Style and positioning
- modern
- premium
- clean
- B2B
- trustworthy
- structured
- conversion-oriented

Avoid childish, playful, overly decorative, or startup-hype styles.

## Priorities
1. Improve clarity of offer
2. Strengthen trust and credibility
3. Improve hero section
4. Improve structure and readability
5. Keep design visually premium
6. Maintain responsive layout
7. Preserve multilingual support

## Design guidance
- clear typography
- strong first screen
- sections should feel structured and premium
- use real business/infrastructure/logistics visuals where appropriate
- no clutter
- no random stylistic changes without reason

## Technical guidance
- preserve project structure unless there is a clear benefit to refactor
- avoid unnecessary dependency bloat
- keep code clean and maintainable
- do not break existing multilingual switching
- ensure all edits are production-ready

## When making changes
Always explain:
1. what was changed
2. why it was changed
3. what files were affected
4. whether anything needs manual review
